## 作品介绍文档

## 一、项目详情及使用方式

### 1.1 项目概述

《道德经》AI互动解读者平台 2.0 是一个融合人工智能技术与中华传统文化的创新型学习交流平台。本项目以《道德经》为核心内容，通过先进的AI技术赋能传统文化教育，为用户提供智能化、个性化、互动式的学习体验。

**项目定位**：打造国内领先的传统文化AI学习平台，让千年智慧在现代科技加持下焕发新生。

### 1.2 核心功能模块

| 功能模块 | 功能描述 |
|---------|---------|
| **AI智能解读** | 基于大语言模型的智能问答系统，支持多轮对话、章节解读、概念解析 |
| **个性化学习路径** | 根据用户水平（初学者/进阶者/研究者）定制专属学习计划 |
| **知识图谱可视化** | 使用D3.js构建《道德经》核心概念关联图谱，支持交互式探索 |
| **共创社区** | 用户交流、心得分享、资源贡献的互动平台 |
| **每日一签** | 游戏化每日抽签功能，增加学习趣味性 |
| **语音合成(TTS)** | 支持AI解读内容的语音朗读，多音色可选 |
| **情绪急救包** | 针对用户情绪状态提供道家智慧疏导 |

### 1.3 使用方式

#### 环境准备
- **Node.js**：v16+ 版本
- **Python**：3.8+ 版本（后端服务）

**2. 启动前端服务**
```bash
cd vue-project

# 前端服务运行在 http://localhost:3000
```

**3. 访问平台**
浏览器打开 http://localhost:3000 即可使用完整功能。

## 二、项目亮点

### 2.1 AI驱动的个性化学习体验

平台核心亮点在于深度融合AI技术，为用户提供真正个性化的《道德经》学习体验：

- **智能问答系统**：用户可就《道德经》任意章节、概念提问，AI基于RAG（检索增强生成）技术提供准确、深入的解读
- **多人格AI对话**：提供"现代学者"、"道家隐士"、"心理疗愈"三种AI人格，满足不同场景需求
- **上下文记忆**：AI能够记住对话历史，支持多轮深入交流

使用D3.js构建的交互式知识图谱是平台的一大特色：

- **交互式探索**：支持缩放、拖拽、点击查看详情
- **章节关联**：每个概念节点关联相关章节，便于深入学习

### 2.3 游戏化学习设计

引入游戏化元素，提升学习趣味性和用户粘性：

- **每日一签**：每日抽取道家智慧签文，获得"功德"（经验值）
- **成就系统**：完成学习任务解锁"初学者"、"思考者"等成就徽章
- **情绪急救包**：针对用户情绪状态提供道家智慧疏导

### 2.4 沉浸式用户体验

- **中国风UI设计**：融合水墨、竹简等传统元素，营造古典氛围
- **响应式设计**：完美适配桌面端、平板、手机等多种设备
- **语音朗读**：支持AI解读内容的语音合成朗读

### 2.5 完整的技术架构

- **前后端分离**：Vue 3 + Node.js/Express 架构
- **现代化技术栈**：TypeScript、Tailwind CSS、Pinia状态管理
- **云原生部署**：支持Vercel等云平台一键部署

## 三、AI测试技术落地使用情况概述

### 3.1 RAG检索增强生成技术

平台核心AI功能基于RAG（Retrieval-Augmented Generation）技术实现：

```typescript
// RAG检索服务核心逻辑
export const intelligentSearch = (query: string): RAGResult => {
  // 预处理查询
  const processedQuery = preprocessQuery(query)
  
  // 多策略检索：模糊搜索 + 章节匹配 + 概念匹配
  const results = {
    fuzzy: searchEngine.search(processedQuery),
    chapter: searchByChapter(processedQuery),
    concept: searchByConcept(processedQuery)
  }
  
  // 合并和排序结果
  const mergedResults = mergeSearchResults(results, processedQuery)
  
  // 构建RAG结果，包含上下文和建议
  return buildRAGResult(mergedResults, query)
}
```

**技术特点**：
- 使用Fuse.js实现模糊搜索，支持容错匹配
- 构建《道德经》知识库，包含81章原文及注释
- 多策略检索融合，提高检索准确率

```typescript
// DeepSeek API 服务
export class DeepSeekService {
  async generateText(prompt: string): Promise<string> {
    const response = await fetch(apiUrl, {

        'Authorization': `Bearer ${this.apiKey}`,
        'Content-Type': 'application/json'

      body: JSON.stringify({
        model: this.model,
        messages: [
          {
            role: 'system',
            content: '你是一位精通《道德经》的道家学者...'
          },
          { role: 'user', content: prompt }
        ],
        max_tokens: this.maxTokens,
        temperature: this.temperature
      })
    })
    // ...
  }
}
```

**技术特点**：
- 支持流式响应，实现打字机效果
- 系统角色设定为"道家学者"，确保回答风格一致
- 支持多种参数调节（temperature、max_tokens等）

### 3.3 语音合成(TTS)技术

集成CosyVoice语音合成服务：

```typescript
// TTS组合式函数
export function useTTS(options: UseTTSOptions = {}) {
  const speak = async (text: string, customOptions?: Partial<TTSOptions>): Promise<void> => {
    const ttsOptions: TTSOptions = {
      voice: customOptions?.voice || currentVoice.value,
      speed: customOptions?.speed || currentSpeed.value,
      emotion: customOptions?.emotion || currentEmotion.value
    }
    
    // 调用语音合成服务
    const result = await cosyvoiceService.textToSpeech(text, ttsOptions)
    
    // 创建音频对象播放
    const audio = new Audio(audioUrl)
    await audio.play()
  }
}
```

**技术特点**：
- 多音色可选（温柔女声、沉稳男声、知性女声）
- 支持语速、情感调节
- 根据AI人格自动匹配音色
