## 🎬 第三部分：AI 解读功能（1.5分钟）

**【画面】** AIInterpretationView 页面

**【操作步骤】**
1. 点击"开始探索"按钮
2. 等待页面加载
3. 在章节选择器中选择"第一章：道可道，非常道"
4. 在输入框中输入："请解释这一章的深层含义"
5. 点击发送按钮
6. 等待 AI 回复生成
7. 展示完整的对话内容

**【讲解要点】**
- 页面布局介绍（对话区、章节选择器、热门问题）
- 章节选择功能（支持 81 章）
- AI 回复的特点（深入浅出、结合现代生活）
- 对话历史保存

---

## 🎬 第四部分：语音合成功能（2分钟）⭐重点

**【画面】** TTSView 页面

**【操作步骤】**
1. 点击"道德经解读"按钮
2. 等待 TTS 页面加载
3. **选择模式**：点击"AI 解读"模式按钮
4. **选择章节**：在下拉框中选择"第一章：道可道，非常道"
5. **选择风格**：选择"通俗易懂"风格
6. **配置语音**：
   - 音色：选择"温柔女声"
   - 语速：选择"适中"
   - 情感：选择"平静"
   - 音量：调整到 80%
7. **点击"生成语音"按钮**（重点强调）
8. **展示自动播放**（等待音频加载和播放）
9. **演示播放器控制**：
   - 点击暂停按钮
   - 点击播放按钮
   - 拖拽进度条到中间位置
   - 点击停止按钮

**【讲解要点】**
- 四种模式介绍（原文朗读、AI 解读、冥想引导、自定义内容）
- 语音配置选项
- **核心亮点**：预生成音频方案
  - 即点即播，无需等待
  - 成本低，体验好
  - 技术实现简单高效
- 播放器控制功能

## 🎬 第六部分：决策助手功能（1分钟）

**【画面】** DecisionHelperView 页面

**【操作步骤】**
1. 点击"决策助手"导航
2. 在决策主题输入框填写："是否应该辞职创业？"
3. 在四个维度填写内容：
   - 阳（优势）：有技术背景、有积蓄
   - 阴（劣势）：缺乏管理经验
   - 天时（机遇）：市场环境有利
   - 地利（威胁）：竞争激烈
4. 点击"生成决策建议"按钮

**【讲解要点】**

## 💡 录制技巧

1. **操作要流畅**：提前熟悉演示流程，避免卡顿
2. **鼠标要明显**：使用鼠标高亮工具，方便观众跟随
3. **关键操作放慢**：重点功能（如语音合成）适当放慢速度
4. **配合讲解**：操作和讲解要同步，不要脱节
5. **适当停顿**：每个功能演示完后稍作停顿，让观众消化

---

## ⚠️ 注意事项

1. **提前启动项目**：确保 `npm run dev` 已经运行
2. **清理浏览器缓存**：确保加载最新代码
3. **关闭无关标签页**：保持界面整洁
4. **测试音频播放**：确保 speakers/耳机正常工作
5. **准备备用方案**：如果某个功能出问题，知道如何跳过
