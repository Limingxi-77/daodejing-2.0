## 一、语音合成功能技术方案

### 1.1 方案选择：预生成音频 vs 实时 TTS

| 对比项 | 预生成音频 | 实时 TTS API |
|-------|-----------|-------------|
| 响应速度 | ⚡ 即点即播 | ⏱️ 需要等待生成 |
| 成本 | 💰 低（一次性生成） | 💸 高（按调用付费） |
| 网络依赖 | 📶 只需下载文件 | 🌐 需要稳定 API 连接 |
| 定制化 | 🎨 固定内容 | ✨ 可动态调整参数 |
| 适用场景 | 固定内容朗读 | 动态内容生成 |

**本项目选择预生成音频方案**，因为道德经内容是固定的，提前生成好音频文件可以提供更好的用户体验。

### 1.2 核心代码实现

**文件位置**：`vue-project/src/views/TTSView.vue`

#### 加载音频文件

```typescript
const generateSpeech = async () => {
  if (!displayText.value.trim()) {
    showToast('请先生成或输入文本内容')
    return
  }

  isGenerating.value = true
  audioStatus.value = 'generating'
  loadingText.value = '正在加载语音...'

  try {
    // 加载预生成的语音文件 video.mp3
    const response = await fetch('/video.mp3')
    if (!response.ok) {
      throw new Error('语音文件加载失败')
    }

    const blob = await response.blob()
    audioBlob.value = blob
    audioUrl.value = URL.createObjectURL(blob)
    audioStatus.value = 'ready'

    // 添加到历史
    history.value.unshift({
      title: `道德经解读语音 ${new Date().toLocaleString()}`,
      time: new Date().toLocaleString(),
      blob: blob
    })

    showToast('语音加载成功！开始播放...')

    // 自动播放音频
    await playAudio()
  } catch (error) {
    console.error('加载失败:', error)
    showToast('语音加载失败，请检查文件是否存在')
  } finally {
    isGenerating.value = false
  }
}
```

#### 播放控制实现

```typescript
const playAudio = async () => {
  if (!audioUrl.value) return

  try {
    const audio = new Audio(audioUrl.value)

    // 监听音频事件
    audio.addEventListener('loadedmetadata', () => {
      duration.value = audio.duration
    })

    audio.addEventListener('timeupdate', () => {
      currentTime.value = audio.currentTime
    })

    audio.addEventListener('ended', () => {
      isPlaying.value = false
      audioStatus.value = 'ready'
      currentTime.value = 0
    })

    audio.addEventListener('play', () => {
      isPlaying.value = true
      audioStatus.value = 'playing'
    })

    audio.addEventListener('pause', () => {
      isPlaying.value = false
      audioStatus.value = 'ready'
    })

    // 设置音量
    audio.volume = voiceConfig.value.volume / 100

    // 开始播放
    await audio.play()

    // 保存音频实例以便控制
    ;(window as any).currentAudio = audio
  } catch (error) {
    console.error('播放失败:', error)
    showToast('音频播放失败')
  }
}
```

### 1.3 文件存储位置

- **开发环境**：`vue-project/public/video.mp3`
- **生产环境**：部署后通过 `/video.mp3` 路径访问
- **静态资源**：Vite 会自动将 public 目录下的文件复制到构建输出目录

### 1.4 播放器功能实现

| 功能 | 实现方法 |
|-----|---------|
| 播放/暂停 | `audio.play()` / `audio.pause()` |
| 停止 | `audio.pause()` + `audio.currentTime = 0` |
| 进度跳转 | `audio.currentTime = targetTime` |
| 音量控制 | `audio.volume = 0-1` |
| 进度显示 | `timeupdate` 事件监听 |
