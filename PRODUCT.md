# Product

## Register

product

> 主寄存器是 product(AI 学习/对话工具是产品核心)。但 Home / About / BusinessPlan
> 三个公共门面将采用 brand 寄存器执行 — 这是评委 30 秒判断"眼前一亮"的舞台。

## Users

**主要用户(C 端)**
- **当代学习者(20-35)**:接受过完整高等教育,对国学有好奇但被传统呈现劝退。他们在"想读经典"和"读不下去"之间,需要一个能托住注意力的入口。
- **修行者 / 长期读者(30-55)**:已经在读《道德经》,需要一个能跨章查证、追问、记录的工具。他们不需要被启蒙,需要被尊重。

**次要用户(B 端)**
- **企业团队 / 教培机构**:把"道学 + AI 解读"作为员工人文素养课、企业文化课的载体。Team 版 / Enterprise 版的受众。

**关键评估者(非用户但决定生死)**
- **传智杯 AI Web 国赛评委**:学术+产业混合背景,30 秒内要看明白"这是什么 / 凭什么不一样 / 凭什么能赚钱"。任何一页都可能成为他点开的第一页。

## Product Purpose

借助 AI 智慧探索老子思想的**现代价值** — 不是把古文翻译成白话,而是让《道德经》能进入一个 2026 年的人面对的真实问题(职业焦虑、人际、决策、伦理),并仍然有效。

成功的样子:用户在某一次具体的犹豫里,真的回到了某一章,并用它做了一个决定。

## Brand Personality

三个词:**安静、克制、当代**。

- **安静**:不喊。不弹窗。不用感叹号。让用户主动靠近,而非被拉拽。
- **克制**:每一像素都在做事。装饰即罪。留白即设计。
- **当代**:道学是 2500 年前的思想,但呈现必须像 2026 年的成熟印刷品 / 当代书籍 / 一线美术馆图录。古意藏在结构与字距里,不藏在装饰素材里。

声音:第二人称、句子短、避免赘词、避免"开启智慧之旅"这类营销语。

## Anti-references

**这个网站绝不能让评委联想到下面任何一类**:

1. **传统国学网站**(国学网 / 中华书局风格站)
   - 红黑金主色 + 龙凤纹边框 + 仿宋楷体 + 印章装饰
   - 太极 / 八卦元素**作为壁纸或重复装饰素材**(以下使用是被允许的:作为唯一的 brand mark 出现的几何精度极高的太极标记;作为真实内容/注释呈现的卦象;不是背景填充、不是 hero 装饰)
2. **中国风 H5 / 古风模板站**
   - 做旧水墨背景 + 折扇/牡丹/仙鹤剪影 + 烫金字 + "国潮"风浮夸装饰
3. **通用 SaaS / AI 落地页**
   - 紫色渐变 + Inter + 浅色 + 三列卡片 + "AI for X" 复制粘贴 hero
4. **教育付费课程站**(得到 / 混沌大学外的模仿者)
   - 密密麻麻信息 + 红色限时优惠 + 教师头像方阵 + 焦虑标语 + 大段销售文案

**正参考(取气质而非抄外观)**:原研哉《白》系列书籍设计 / 朱赢椿书籍排版 / Are.na 长文版面 / NYT 长读 / Aeon Magazine 文章页 / 一些当代美术馆图录(如 Mori Art Museum 出版物)。

## Design Principles

1. **留白即叙事** — 不填满。让用户的视线有节奏地停下、再走。
2. **典则不仿古** — 保留古典的层级骨架(章/节/段/注),拒绝任何"做旧"装饰。
3. **字承道意** — 道学的重量靠字号、字重、字距、行高承载,不靠太极图 / 印章 / 仿宋。
4. **功能高于氛围** — 美学不能让工具变难用。AI 对话、学习路径、章节解读必须依旧 30 秒内可达可用。任何新增/删改既有功能必须先确认。
5. **三十秒可读** — Home / About / BusinessPlan 三页,任意一页打开后 30 秒内,陌生人能说清"这是干什么的、凭什么不一样"。

## Accessibility & Inclusion

务实基线,**不追求合规数值,追求"读得下去"**:

- 文本对比度肉眼可读即可(不强制 WCAG AA 数值)
- 键盘 Tab 顺序合理、focus 状态可见
- 支持 `prefers-reduced-motion`:所有动画在系统设置下自动降级为 fade-in 或无动画
- 中文 only(项目无 i18n 计划)
- 桌面优先 + 移动可用(竞赛 demo 主要在桌面投屏,不强求 200% 缩放守版)
